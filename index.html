Looking to join?

I am looking for PhD students for the Fall 2024 application / Fall 2025 start cycle, particularly those interested in AI for science and the application of machine learning methods to problems in biology, chemistry, and biochemistry.

You don't need to have any existing biology or chemistry background, although it's always a plus! Our lab predominately publishes in machine learning venues like ICML, NeurIPS, AISTATS, and ICLR, but we do collaborate quite closely with several folks across the physical sciences, and especially at Penn Medicine.

If you are interested, apply to the CIS PhD Program directly, but do mention me in your application!

Software

Geoff and I founded the GPyTorch project, which aims to implement Gaussian processes in a modular package with strong GPU acceleration. It is deeply embedded in the PyTorch ecosystem, and makes designing complicated models like deep kernel learning both easy to implement and highly efficient. It departs significantly from existing Gaussian process libraries, in that it makes use of modern numerical linear algebra techniques like linear conjugate gradients to perform the fundamental operations required for inference, rather than standard Cholesky based approaches.
